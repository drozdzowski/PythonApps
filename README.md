# PythonApps Repository

This repository contains two separate Python GUI projects:

- `TicTacToeGUI`: A GUI-based Tic Tac Toe game using Tkinter.
- `RNGeneratorGUI`: A GUI-based random number generator using Tkinter.

Each project has its own folder with a `main.py` and `README.md` for instructions.

## How to Run

Navigate to the desired project folder and follow the instructions in its README.md.
