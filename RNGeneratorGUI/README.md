# GUI Random Number Generator Python Project

This is a simple GUI-based random number generator implemented in Python using Tkinter.

## How to Run

1. Make sure you have Python installed.
2. Open a terminal in this directory.
3. Run:

    python main.py

## Features
- Specify min and max range
- Mouse support for generating numbers
- Input validation
- Easy to extend for more features

## File Structure
- `main.py`: Main GUI application logic
