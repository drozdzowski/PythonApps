# Soccer Score Predictor

This app pulls data about the English Premier League from the internet and predicts results for upcoming games based on player and team past performance.

## Features
- Fetches latest EPL data (teams, players, fixtures, results)
- Analyzes team and player statistics
- Predicts scores for upcoming matches
- Simple GUI for user interaction

## How to Run
1. Make sure you have Python 3 installed.
2. Install required packages:
   pip install requests beautifulsoup4 pandas scikit-learn
3. Run the app:
   python main.py
