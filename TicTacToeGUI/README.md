# GUI Tic Tac Toe Python Project

This is a simple GUI-based Tic Tac Toe game implemented in Python using Tkinter.

## How to Run

1. Make sure you have Python installed.
2. Open a terminal in this directory.
3. Run:

    python main.py

## Features
- Two-player mode (X and O)
- Mouse support for moves
- Win and draw detection
- Easy to extend for AI or improved GUI

## File Structure
- `main.py`: Main GUI game logic
